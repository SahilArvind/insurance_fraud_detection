About This Project

Overview

This project uses machine learning to detect fraudulent insurance claims. A Random Forest classifier is trained on a dataset of 1,000 insurance claims with 28 features covering policy details, insured demographics, incident information, and claim amounts.

Tech Stack

  • Python — Core language
  • scikit-learn — Model training and evaluation
  • pandas — Data preprocessing
  • Flask — Web application framework
  • joblib — Model serialization

Features Used (28)

Age, policy state, CSL, deductible, annual premium, umbrella limit, sex, education level, occupation, hobbies, relationship, capital gains/losses, incident type, collision type, severity, authorities contacted, incident state/city/hour, vehicles involved, property damage, bodily injuries, witnesses, police report availability, total claim amount, auto make, and auto year.
